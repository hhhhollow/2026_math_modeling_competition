# 论文文件夹

## 编译方式

需要 LaTeX 发行版（推荐 TeX Live 或 MacTeX），且必须用 **xelatex**（中文支持）：

```bash
cd 论文
xelatex main.tex
xelatex main.tex   # 跑两次以解决目录与交叉引用
```

或用 latexmk：

```bash
latexmk -xelatex main.tex
```
